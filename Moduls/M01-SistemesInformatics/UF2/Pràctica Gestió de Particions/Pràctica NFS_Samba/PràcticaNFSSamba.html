7.- Indica què ens aporta Samba que no tingui ja NFSv3.

Amb Samba es poden compartir carpetes entre màquines amb sistemes Unix i Windows, a més de poder compartir impressores. Samba permet més configuració d’accés dels usuaris que no pas NFSv3.

8.- Quina diferència hi ha entre Samba i CIFS?

Samba permet compartir directoris i fitxers entre diferents sistemes operatius. CIFS permet compartir directoris i fitxers entre sistemes amb Windows.
